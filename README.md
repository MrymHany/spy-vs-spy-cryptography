# Spy Vs Spy: Cryptography Challenge 🔐🕵️

An interactive, gamified platform designed to teach the fundamentals of symmetric encryption through competitive gameplay.

## ✨ Key Features
- **Dual Algorithms:** Implements both **AES** and **DES** encryption/decryption logic.
- **Gamified Experience:** Includes timed rounds, hint systems, and scoreboard tracking.
- **UI/UX:** Features a professional graphical interface built with Java Swing.

## 🛠️ Tech Stack
- **Language:** Java
- **Cryptography:** Java Cryptography Extension (`javax.crypto`) using ECB mode.
- **GUI:** Java Swing.

## 🛡️ Security Concepts
Players learn about key lengths (56-bit to 256-bit), the difference between block ciphers, and the importance of key management.
